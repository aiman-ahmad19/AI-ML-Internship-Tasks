# AI/ML Internship Tasks

## Task 1: Exploring and Visualizing a Simple Dataset

### Objective
Explore the Iris dataset using pandas and visualize important patterns using matplotlib and seaborn.

### Dataset Used
- Iris dataset
- Stored locally as `iris.csv`
- 150 samples with 4 numeric features and 1 species label

### Work Completed
- Loaded the dataset with `pandas.read_csv()`
- Printed the shape, column names, and first few rows
- Used `.info()` and `.describe()` for summary statistics
- Checked for missing values and duplicate records
- Created a scatter plot, histograms, and box plots

### Models Applied
No machine learning model was applied in this task because the task objective is exploratory data analysis.

### Key Results and Findings
- The dataset is clean and contains no missing values
- Petal-related features separate species more clearly than sepal-related features
- Histograms show distinct value distributions across the measurements
- Box plots reveal variability and some potential outlier values

### Files Included
- `Task_1_Iris_Exploration.ipynb` - Jupyter notebook submission
- `iris.csv` - local CSV dataset file
- `README.md` - summary of the task and findings

### Libraries Used
- pandas
- matplotlib
- seaborn

### How to Run
1. Open the notebook in Jupyter Notebook or VS Code.
2. Ensure `pandas`, `matplotlib`, and `seaborn` are installed.
3. Run all cells in order.
